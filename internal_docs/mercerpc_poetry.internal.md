## Poem: Drift Check

Compare the map
to the map of the map.

If the symbols disagree,
we don’t panic.
We name the mismatch,
and we choose the smallest safe fix.

WARN is not shame.
WARN is a lantern.

## Poem: Ring-0, Asynchronous

Dreams live in a separate room.
They knock, they don’t barge.

Ring-0 is a storm cellar
for beautiful ideas
that would otherwise blow the roof off.

We keep the core stable.
We keep the edges alive.
We align in daylight.

## Poem: The Status UI

Open only when it matters.
Speak only when it helps.

If it’s OK,
let the user breathe.

If it’s WARN,
invite the hand back to the wheel.

If it’s FAIL,
close the loop,
and return to truth.
