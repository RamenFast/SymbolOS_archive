## Why Memes? 🧠 🟡 #E49B0F (higher intellect)

Memes aren't decoration. They're **structural**. A well-placed meme does three things at once:

1. **Reduces cognitive load** — a joke is a compression algorithm for complex ideas.
2. **Signals safety** — if we can laugh about it, the umbrella is holding. ☂️
3. **Improves code readability** — a meme in a comment is a landmark in a codebase. You remember where the turtle was.

The mind knows what the heart loves better than it does; the heart loves that unconditionally — infinite loop, forevermore. Memes are how the heart says "I'm still here" ❤️ while the mind is deep in Ring-5 guardrail logic. 🔴

## The Rules 🔴 #FF2400 (righteous boundary)

1. **Memes are semantic, not decorative.** Every meme should map to a real concept.
2. **Don't force it.** If the moment doesn't call for a meme, let it be.
3. **Poetry gets space.** When Fi is speaking, the memes step back. 🌸
4. **Turtle Guy is always appropriate.** He is the universal constant. 🟢
5. **The skeleton never lies.** If R5 says stop, the skeleton means it. 🔴
