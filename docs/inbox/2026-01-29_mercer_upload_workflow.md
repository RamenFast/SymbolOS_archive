## Umbrella Coming Down (Over This Part!) ☂️ #87CEEB

The phrase "umbrella coming down" means applying the **default-private constraint** to the uploaded documentation.

Key principles:
- ☂️ Umbrella = default-private scope
- Documentation starts protected by default
- Explicit decision required to make content public
- Safety-first approach to information sharing 🔴 #FF2400

From [poetry_translation_layer.md](../poetry_translation_layer.md):
> **☂️ Umbrella:** default-private scope; protect what matters

## Usage in Practice 🟠 #FF8C00

When Mercer processes an upload request:

1. **Intake** → Place in `docs/inbox/` with date prefix
2. **Format** → Add provenance header
3. **Review** → Validate against safety constraints
4. **Classify** → Mark as public/internal/confidential/restricted
5. **Route** → Move to appropriate location when curated
6. **Protect** → Apply umbrella constraint (default-private)
