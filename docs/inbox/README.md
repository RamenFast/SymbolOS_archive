# Docs Inbox

Drop incoming notes here before curating them into the main docs.

Conventions:

- Prefix filenames with date: `YYYY-MM-DD_...`
- Prefer Markdown when possible.
- Add a short provenance header at the top of each file:
  - Source (Drive/GitHub/etc)
  - Owner
  - Date
  - Classification (public/internal/confidential/restricted)
