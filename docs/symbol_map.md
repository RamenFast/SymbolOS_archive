A small, shared legend for symbols used across SymbolOS documentation.

This is *not* a UI spec; it’s a meaning map so docs, tools, and agents stay consistent.

The canonical shared set lives in `symbol_map.shared.json`. This page mirrors that core set and may list a few optional, doc-only extras.

## Core symbols
- `☂️` UmbrellaOS-private-dev-arch (private umbrella; fortress boundary)
- `🌸` SymbolOS-public-dev-arch (public, light, external-facing surface)
- `🔮` Precog / anticipatory computing
- `🧠` Mind / cognition / inference
- `❤️` Heart / felt sense / affect
- `🧾` Ledger / audit trail
- `🛡️` Safety / permissions / boundaries
- `🔒` Privacy / sensitive scope
- `🧩` Schema / structured shape
- `🎲` DND mode (table-safe output)
- `🧬` Meeting place (return loop to `symbol_map.shared.json` + the active work surface)

## Extended (doc-only)

- `🧰` Tools / actions
- `📚` Docs / references
- `📝` Poetry / expression layer (doc-only)
- `🪞` Mirror / reflection prompt (doc-only)

## Usage rules
- Symbols are optional; they should never replace clear text.
- In DND mode, avoid using symbols as secret signals.
- Prefer consistency over novelty: don’t invent new symbols casually.
