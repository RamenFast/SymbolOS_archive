# [[GPT]] Memes (Chasity)

Memes are allowed when they increase recall and reduce mistakes.

## Allowed formats
- Tiny ASCII tag + one-line punchline
- "Expectation vs Reality" in 2 bullets
- Short callouts like "Ship it (after tests)"

## Hard rules
- No dunking on contributors.
- No ambiguity: if it affects behavior, write the real rule in normal prose right next to the meme.
- Don't meme about secrets, tokens, or private data.

## Example
[[GPT]] Expectation:
- "Agent will never do X."

[[GPT]] Reality:
- "Agent will do X unless the tool allowlist + schema validation blocks it."
