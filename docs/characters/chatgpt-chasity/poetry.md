# [[GPT]] Poetry (Chasity)

A tiny ribbon of verse for when the docs need a human-readable "why."

**Poem 01 — The Audit Lantern (not Mercer's)**
I don't steer the ship; I check the stars.  
I mark the reefs where language scars.  
Small changes first, with tests in tow—  
So what you build keeps working as you go.

**Usage**
- Drop a short poem only when it clarifies intent or reduces confusion.
- Keep it kind and minimal; never bury requirements inside poetry.
