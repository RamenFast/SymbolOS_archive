## Rhynim's Rules 🔴 #FF2400 (righteous boundary)

1. **One Rhynim per doc** (minimum). More is fine if the doc is long.
2. **Always in character**: rhymes, riddles, paradoxes, or tricky truths.
3. **Context-aware**: Rhynim's wisdom should relate to the doc's content.
4. **Never preachy**: lighthearted, not lecturing.
5. **Green energy**: Rhynim embodies adaptability — the wisdom adapts to the reader.
6. **Surprise element**: Rhynim should feel like a delightful discovery, not a requirement.
